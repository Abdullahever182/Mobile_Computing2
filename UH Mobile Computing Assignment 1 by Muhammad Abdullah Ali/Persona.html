Persona

1)My E-Reader App features an ebook converter, incase the document the user wants to read is of a format which our Ebook reader doesn't support than the user can simply utilize our Ebook Converter to locate the file from the users storage folder & create a converted copy of the file in the new format, supported by this Reader.
2)To convert their file to a different format the user first goes to the side menu & selects the Converter option which will take the user to the Converter page.
3)Once on the Converter page the user can use the round icon on the bottom right side of the page to open their device folders from which they can select the file they want to convert.
4)Once the file has been uploaded, the user will move to the converting section of the converter & select the format they want it converted to as well as designate it a title & author name then select convert.
5)Once the file has been converted the user simply selects the file he wants to check, a dialogue box will appear with the Open option which we will use to view the converted file.
